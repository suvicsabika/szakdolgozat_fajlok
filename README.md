# Django:

## 1. Érdemes létrehozni egy virtuális környezetet a Python függőségeknek:
```bash
python -m venv myenv
```

### Aktiválás:

Windows:
```bash
myenv\Scripts\activate
```
Linux:
```bash
source myenv/bin/activate
```

## 2. Django telepítés:
```python
pip install django
pip install djangorestframework django-cors-headers
```
